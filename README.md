# NEXIST Unity Optics Website

A premium eyewear homepage built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui.

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

```bash
pnpm lint
pnpm build
```
